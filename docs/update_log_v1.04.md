# Update Log v1.04

## Date
- 2026-06-10 (KST)

## Scope
- 라운드/경기 정렬 안정화
- 경기 결과 입력 UX 개선

## Changes
- `round` 정렬을 자연정렬 기준으로 통일
  - 예: `1, 2, 3, ... 10, 11 ...`
- 경기 목록 정렬 기준 고정
  - `round -> court -> match_id(생성순)`
  - 같은 라운드/코트 안에서 먼저 생성된 경기가 위에 유지되도록 개선
- 경기입력 페이지에서 저장 후 순서가 바뀌는 현상 완화
  - 정렬 기준 고정으로 입력/저장 시 리스트 순서가 안정적으로 유지됨
- 경기입력 페이지 하단에 `일괄 저장` 버튼 추가
  - 현재 화면에 보이는 경기(필터 반영)의 점수를 한 번에 저장

## Note
- 일괄 저장은 현재 입력창 값 기준으로 저장됩니다.
- 미입력 경기 기본값이 0으로 보이는 상태에서 저장하면 0점이 저장될 수 있습니다.
