# Update Log v1.03

## Date
- 2026-06-09 (KST)

## Scope
- 대회 결과 페이지 UI/엑셀 내보내기 개선

## Changes
- 순위표 컬럼 재정렬 및 명칭 정리
  - `승리수 -> 승`, `총점 -> 승점`, `추가점수(합산) -> 보너스`
  - 컬럼 순서: `순위, 이름, 경기수, 승점, 승, 무, 패, 득실차, 승률, 보너스`
- `승률` 컬럼 추가
  - 계산식: `승 / 경기수` (0경기면 0%)
- 엑셀 내보내기(XLSX) 확장
  - `순위표` 시트
  - `승점계산방식` 시트
  - `경기결과` 시트 (점수 입력 완료 경기만)
- `경기결과` 시트/표 컬럼 구성
  - `라운드, 코트, 팀1, 팀1 점수, 팀1 결과, 팀2 결과, 팀2 점수, 팀2`
  - 결과값 자동 산출: `승/패/무`
- 대회 날짜 표시 및 파일명 postfix 추가
  - 페이지 상단에 선택된 대회 날짜 표시
  - 파일명: `..._순위표_YYMMDD.xlsx`
  - 날짜 비정상/없음일 때 postfix는 `00000`
- 용어 정리
  - `현재 점수 계산 방식 확인 -> 승점 계산 방식`
  - 순위표 바로 아래에 `승점 계산 방식` 토글 배치

## Note
- 본 로그는 `대회 결과` 중심 변경사항만 간략 정리했습니다.

## 통계 페이지 추가 업데이트
- `연도별 랭킹포인트`를 `시즌별 랭킹포인트`로 변경
- 시즌별 랭킹포인트 차트를 연도 누적 기준으로 개선
- 연도별 동률 순위 라벨을 병합 표기 (`n위 이름1,이름2`)
- 범례(선수 리스트) 정렬을 최종 시즌 랭킹 순서로 정렬
- `대회별 통계` 섹션 제목/구조 정리 (불필요한 구분선 제거)
- 대회 선택 후 `선택 대회 날짜` 표시 추가
- 용어 변경:
  - `총점 비교` -> `승점 비교`
  - `승률 (%)` -> `세트 승률 (%)`
  - `현재 점수 계산 방식 확인` -> `승점 계산 방식` (토글 없이 본문 표시)
- 선수별 종합 비교 레이더 차트 지표 변경 및 0~100 정규화 적용
- 레이더 차트 색상 5종 고정 적용 + TOP3(메달) 텍스트 표시 추가
